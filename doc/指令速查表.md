# 指令速查表

> 串口：115200, 8N1 | 格式：`X:数值` 或直接发送命令（回车结尾）

---

## 一、参数设置

| 命令 | 格式 | 说明 | 自动保存 |
|:----:|------|------|:--------:|
| **T** | `T:数值` | 大循环次数 | ✗ |
| **H** | `H:数值` | 小循环次数（工位间往返次数） | ✗ |
| **F** | `F:数值` | 运行最大速度限制 | ✓ |
| **R** | `R:数值` | 最大占空比 | ✓ |
| **V** | `V:数值` | 最后匀速进孔速度 | ✓ |
| **S** | `S:数值` | 慢速进孔脉冲数 | ✓ |
| **P** | `P:1~5` | 拍照孔位（到达该工位时拍照） | ✗ |
| **E** | `E:数值` | 光电检测限制脉冲数 | ✓ |

---

## 二、控制命令

| 命令 | 说明 |
|:----:|------|
| `ELE` | 光电信号检测（读 PE4 电平） |
| `OP` | 拍照测试（触发一次拍照） |
| `CLC` | 报警清除（清 error 标志） |
| `D0` | 电机初始化 |
| `HAL` | 霍尔位置检测（读 3 路霍尔状态） |
| `RED` | 实时读取编码器位置（持续输出，需复位退出） |
| `RTF` | 正转（前进一个工位） |
| `RTB` | 反转（后退一个工位） |
| `TIM3` | TIM4 计时测试（5s） |
| `RING` | 外部中断上升沿检测测试 |
| `RTS` | 恢复初始位置（停止电机，清零目标位） |
| `RES` | 保存参数并复位系统 |

## 四、测试模式

| 命令 | 说明 |
|:----:|------|
| `gnn` | **磨合测试** — 5 站循环往返，每站间往返 loop2 次，大循环 loop1 次 |
| `goo` | **自动测试** — 5 站循环，每站间往返 loop2 次，每轮拍照 |
| `gff` | **正转拍照测试** — 持续正转，到达 P 孔位时拍照（无限循环） |
| `gmm` | **反转拍照测试** — 持续反转，到达 P 孔位时拍照（无限循环） |

## 六、查询与调试

| 命令 | 说明 |
|:----:|------|
| `PARA` | 显示所有当前参数 |
| `ALARM` | 查询报警代码 |
| `DBG` | 开始采集正向运动波形数据（位置+速度，最多 300 点，25ms 间隔） |
| `DUMP` | 打印 DBG 采集的数据（CSV 格式） |

---

## 七、报警代码

| 代码 | 含义 |
|:----:|------|
| `0` | 无报警 |
| `1` | 24V 断电 |
| `2` | 电机限位超时 |
| `3` | 光电检测异常 |
| `4` | 运动超限 |
| `5~6` | 运行超时 |
| `7` | 工位到达失败 |
| `10` | 堵转超时 |

> 报警处理：`ALARM` 查询 → 排查硬件 → `CLC` 清除 → `D0` 重新初始化
