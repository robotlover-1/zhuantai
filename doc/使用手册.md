# 平式步进电机控制系统 — 使用手册

## 1. 概述

本项目基于 **STM32F407ZG** 微控制器，控制平式步进电机在 5 个工位间精确定位，支持 PID 闭环控制、光电/霍尔位置检测、拍照触发、参数 Flash 存储、磨合测试等功能。

### 1.1 硬件资源

| 外设 | 功能 | 参数 |
|------|------|------|
| **MCU** | STM32F407ZG | 168MHz |
| **TIM1** | 电机 PWM 驱动 (CH1 正转, CH4 反转) | ~4.24kHz |
| **TIM3** | 编码器位置计数 | 84MHz |
| **TIM4** | 磨合运行计时 | 1s 中断 |
| **TIM5** | 超时/堵转检测 | 0.5s 中断 |
| **TIM6** | PID 控制周期 | 1ms 中断 |
| **UART5** | 主串口（命令交互，PC 串口终端） | 115200 baud |
| **UART4** | 从串口（与拍照设备通信） | 9600 baud |
| **I2C2** | (预留) | - |
| **SPI2** | (预留) | - |
| **FSMC** | (预留) | - |

### 1.2 传感器

| 信号 | 引脚 | 说明 |
|------|------|------|
| 光电信号 (ELE) | PE4 | 球式光电信号输入 |
| 平式光电 (IO) | PF9 | 双向光耦输入 |
| 霍尔 1 (HAL1) | PE5 | 工位检测 |
| 霍尔 2 (HAL2) | PE6 | 工位检测 |
| 霍尔 3 (HAL3) | PC13 | 工位检测 |
| 24V 输入 1 (IN1) | PF6 | 复位信号检测（上升沿清除报警） |
| 24V 输入 2 (IN2) | PF7 | 急停信号检测（下降沿紧急停止） |
| 回零信号 (HOME) | PF8 | 步进回零完成信号 |

### 1.3 输出信号

| 信号 | 引脚 | 说明 |
|------|------|------|
| 关断 (SHUTDOWN) | PC1 | 开漏输出，控制电机使能 |
| 到位 (DONE) | PC3 | 推挽输出，到位脉冲 |
| 报警 (ALARM) | PC4 | 推挽输出，报警脉冲 |
| 拍照 (PHOTO) | PC5 | 推挽输出，触发相机拍照 |

## 2. 连接方式

使用 **USB 转 TTL 串口模块** 连接 PC 与设备：

| 串口模块 | 设备端 (UART5) |
|----------|----------------|
| TX | PC12 (UART5 RX) |
| RX | PD2 (UART5 TX) |
| GND | GND |

> **波特率**: 115200, 8N1, 无流控

打开串口终端软件（如 **SecureCRT**, **Putty**, **串口助手** 等），连接对应 COM 口，设置波特率 115200。

## 3. 命令列表

所有命令通过串口终端发送，以回车（`\r\n`）结尾。

### 3.1 参数设置类

发送格式：`X:数值`（如 `T:10`）

| 命令 | 格式示例 | 参数范围 | 说明 | 自动保存到 Flash |
|------|----------|----------|------|:---:|
| `T:数值` | `T:10` | 整数 | 设置大循环次数 | ✗ |
| `H:数值` | `H:5` | 整数 | 设置小循环次数（往返次数） | ✗ |
| `F:数值` | `F:1000` | 整数 | 运行最大速度限制 | ✓ |
| `R:数值` | `R:2500` | 整数 | 最大占空比 | ✓ |
| `V:数值` | `V:100` | 整数 | 最后匀速进孔速度 | ✓ |
| `S:数值` | `S:14000` | 整数 | 慢速进孔脉冲数 | ✓ |
| `P:数值` | `P:1` | 1~5 | 拍照孔位（到达该工位时触发拍照） | ✗ |
| `E:数值` | `E:38000` | 整数 | 光电检测限制脉冲数 | ✓ |

#### 测试步骤示例

```
发送: T:3         → 返回: loop1:3          (设置大循环为3次)
发送: H:2         → 返回: loop2:2          (设置小循环为2次)
发送: F:1200      → 返回: 运行中最大速度限制:1200
发送: R:3000      → 返回: 最大占空比设置:3000
发送: V:80        → 返回: 最后匀速进孔的速度:80
发送: S:15000     → 返回: 慢速运行进孔脉冲数:15000
发送: P:3         → 返回: 拍照孔位:3        (到达3号工位时拍照)
发送: E:40000     → 返回: 光电检测限制脉冲数:40000
```

### 3.2 控制类命令

| 命令 | 说明 | 返回示例 |
|------|------|----------|
| `ELE` | 光电信号检测（读取 PE4 电平） | `光电信号 1` |
| `OP` | 拍照测试（触发一次拍照信号） | `进入拍照` |
| `CLC` | 报警清除（清除 error 标志） | (无输出) |
| `D0` | 电机初始化 | — |
| `HAL` | 霍尔位置检测（读取 3 路霍尔状态） | `station: 1` + `HAL1: 0` `HAL2: 1` `HAL3: 0` |
| `RED` | 实时读取编码器位置（持续输出，需重启） | `Pos: 12345` (每 50ms 输出一次) |
| `RTF` | 正转（前进一个工位） | 电机正转到下一工位 |
| `RTB` | 反转（后退一个工位） | 电机反转到上一工位 |
| `TIM3` | TIM4 计时测试（启动 TIM4 计时 5s） | `time_run: 5` |
| `RING` | 外部中断上升沿检测测试 | `上升沿检测测试！` |
| `RTS` | 恢复初始位置（停止电机，清零目标位置） | — |
| `RES` | 复位系统（保存参数到 Flash 后重启） | — |

#### 测试步骤示例

```
发送: ELE         → 返回: 光电信号 1         (光电传感器触发中)
发送: OP          → 返回: 进入拍照           (PC5输出拍照脉冲)
发送: HAL         → 返回: station: 2         (当前在2号工位)
                       HAL1: 0
                       HAL2: 1
                       HAL3: 0
发送: CLC         → 返回: (清除了报警)
发送: D0          → 返回: (电机初始化)
发送: RTF         → 返回: 电机正转前进到下一工位
发送: RTB         → 返回: 电机反转到上一工位
发送: RTS         → 返回: (电机停止，回到初始位置)
```

| `L2` | 跳转到工位 2 |
| `L3` | 跳转到工位 3 |

### 3.4 测试模式命令

| 命令 | 说明 |
|------|------|
| `gnn` | **磨合测试**：5 站循环往返运动，每站间往返 loop2 次，大循环 loop1 次 |
| `goo` | **自动测试**：5 站间循环测试，每站间往返 loop2 次，每轮拍照 |
| `gff` | **正转拍照测试**：持续正转，到达拍照孔位(P)时触发拍照 |
| `gmm` | **反转拍照测试**：持续反转，到达拍照孔位(P)时触发拍照 |

#### 测试步骤示例

**磨合测试 (gnn)：**
```
1. 先设置参数:
   T:2    → 大循环2次
   H:1    → 小循环1次
2. 发送: gnn
3. 观察: 电机在5站间正转往返运动，控制台输出每站循环进度
```

**自动测试 (goo)：**
```
1. 先设置参数:
   T:1    → 大循环1次
   H:1    → 小循环1次
   P:3    → 在3号工位拍照
2. 发送: goo
3. 观察: 电机5站间往返，到达工位3时触发拍照
```

**正转拍照测试 (gff)：**
```
1. 先设置拍照孔位: P:3
2. 发送: gff
3. 观察: 电机持续正转，每次到达工位3时拍照
   (该测试为无限循环，需断电或复位退出)
```

### 3.5 工位往返测试命令

| 命令 | 说明 |

## 4. 报警代码说明

| 报警代码 | 含义 | 可能原因 |
|:--------:|------|----------|
| `0` | 无报警 | 正常工作 |
| `1` | 24V 断电 | 24V 电源断开或 IN2 信号丢失 |
| `2` | 电机限位超时 | 运动超过限制时间未到达 |
| `3` | 光电检测异常 | 光电传感器未触发或信号异常 |
| `4` | 运动超限 | 编码器读数超过极限范围 |
| `5`-`6` | 运行超时 | 电机运行超过设定的超时时间 |
| `7` | 工位到达失败 | 电机停止后霍尔传感器未检测到有效工位 |
| `10` | 堵转超时 | 电机堵转，编码器计数超出目标值+5000 |

### 报警处理流程

1. 发送 `ALARM` 查询报警代码
2. 排查对应硬件问题（电源/传感器/机械卡死）
3. 发送 `CLC` 清除报警
4. 发送 `D0` 重新初始化电机
5. 重新测试

## 5. 完整测试流程

### 5.1 上电自检

```
1. 连接串口终端 (115200, 8N1)
2. 给设备上电
3. 观察输出:
   - RTC 初始化信息（当前时间）
   - 参数加载信息
   - 电机初始化状态
4. 发送: PARA     → 查看当前参数
5. 发送: HAL      → 确认霍尔传感器读数正常 (1~5)
6. 发送: ELE      → 确认光电传感器正常
```

### 5.2 手动运动测试

```
1. 发送: D0       → 电机初始化
2. 发送: L1       → 跳转到工位1
3. 发送: L2       → 跳转到工位2
4. 发送: L3       → 跳转到工位3
5. 发送: L4       → 跳转到工位4
6. 发送: L5       → 跳转到工位5
7. 发送: RTF      → 正转一个工位
8. 发送: RTB      → 反转一个工位
```

### 5.3 参数调试

```
1. 发送: F:1500   → 调整最大速度
2. 发送: R:3000   → 调整最大占空比
3. 发送: S:14000  → 调整慢速进孔脉冲
4. 发送: PARA     → 确认参数已保存
5. 发送: L2       → 测试运动效果
```

### 5.4 自动测试

```
1. 设置参数: T:2  H:1
2. 发送: gnn      → 磨合测试（2大循环，每站间往返1次）
3. 等待测试完成
4. 发送: PARA     → 查看最终状态
```

### 5.5 波形调试（可选）

```
1. 发送: DBG      → 开始采集波形
2. 发送: RTF      → 启动一次正向运动
3. 等待运动结束
4. 发送: DUMP     → 导出 CSV 数据
5. 将 CSV 数据粘贴到 Excel 或 Python 中绘图分析
```

## 6. 注意事项

1. **急停保护**：设备检测到 IN2 信号下降沿（24V 断电）时会自动停止电机并置 error=1。恢复 24V 后需发送 `CLC` 清除报警。
2. **复位信号**：IN1 信号上升沿会自动清除 error 标志（软复位）。
3. **参数保存**：带 ✓ 标记的参数在设置时会自动保存到 Flash（Sector 7, 0x08060000），断电不丢失。发送 `RES` 也会先保存参数再复位。
4. **安全操作**：
   - 发送 `RED` 后会进入无限循环输出位置信息，需重新上电或复位才可退出
   - 发送 `gff`/`gmm` 后会进入无限循环测试，需复位退出
   - 发送 `RES` 会立即复位 MCU
5. **首次使用**：若 Flash 中无有效参数（写入标志无效），会自动使用代码中的默认值。
6. **RTC**：系统启动时从编译时间初始化 RTC，断电后需重新设置（当前代码中为编译时自动设置）。
