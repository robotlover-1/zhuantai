# 速度环条件积分抗饱和 — 消除高速→低速切换震颤

日期: 2026-06-30 | 状态: 已实现

## 问题描述

电机从高速区(rf=1)切换到低速区(rf=2/3)的瞬间，发生一次卡顿震颤。运行时数据显示切换瞬间速度误差可达 500~900 单位，增量式速度 PID 的累加器 `ActualValue_S` 在瞬态大误差期间快速累积，进入低速段后需要十几个周期才能释放回正常水平，期间 PWM 过冲造成震颤。

## 根因分析

- 当前速度环参数: SK=1.5, SI=1.0, SD=0
- 切换瞬间 SpdS 从 ~-239 跳到 -160，但实际速度 ~-1100，误差 ~+900
- 增量式 PID 每周期累加 `I * Error = 1.0 * 900 = 900` 到 `ActualValue_S`
- 累加器被推到远超低速稳态所需的值，释放过程导致 PWM 反复过冲

## 方案

条件积分(conditional integration) — 误差大时降低积分贡献，误差小时恢复正常。

### 实现位置

`Core/Src/pid.c` — `increment_pid_speed()` 函数

### 积分衰减曲线

| 误差范围 | 积分强度 | 说明 |
|----------|----------|------|
| > 400 | 30% | 瞬态期间积分几乎冻结 |
| 200 ~ 400 | 30% → 100% | 线性过渡，无突变 |
| < 200 | 100% | 稳态满积分，精度不变 |

### 代码变更

```c
// 条件积分抗饱和: 误差>400时积分衰减到30%, 200~400线性过渡, <200满积分
float i_scale = 1.0f;
float abs_err = (Error_S > 0.0f) ? Error_S : -Error_S;
if (abs_err > 400.0f)
    i_scale = 0.3f;
else if (abs_err > 200.0f)
    i_scale = 0.3f + 0.7f * (400.0f - abs_err) / 200.0f;

ActualValue_S += (Proportion_S * (Error_S - LastError_S))
               + (i_scale * Integral_S * Error_S)       // 积分项乘衰减系数
               + (Derivative_S * (Error_S - 2*LastError_S + PrevError_S));
```

## 影响范围

- 仅 `pid.c` 一个文件，`increment_pid_speed()` 一个函数
- 稳态运行时(`|Error| < 200`)行为与原代码完全一致
- 只影响瞬态大误差场景的积分累积速度

## 验证方法

1. 编译烧录固件
2. 发送 `DP:25` 开启调试打印
3. 执行孔位切换（如 `L2`），观察 `rf=1→3` 切换瞬间的 PWM 和 spd 变化
4. 预期：切换后 PWM 平滑过渡，spd 单调收敛到设定点，无反复穿越
